Click start to hear and see some Oscilloscope music

These graphics are created by combining stereo waveforms of specially created music. The original artist is Jerobeam Fenderson (oscilloscopemusic.com) who creates music that can be visualised through an oscilloscope.

As I don't have an oscilloscope, I thought I would code a JavaScript player that did the same thing.

I'm using the HTML5 Audio API to the play and interpret his tracks, and running the visualisation through canvas.

This demo is using the track "Planets". I highly recommend you check the whole album which you can buy complete with videos. The results through the oscilloscope have a smoother feel.
