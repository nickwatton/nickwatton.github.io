PixelCam

This is a webcam toy

The webcam feed is read and colour / brightness points analysed.
This data is used to render various pixels, rather than the video feed.

Controls:

Two yellow rectangles.

  • Left - Inverts image brightness
  • Right - Changes pixel effect
